# Combo launch

Run `zapret-all.cmd` as Administrator.

It starts:

- Flowseal TG WS Proxy for Telegram Desktop.
- Flowseal zapret with the strategy saved in `utils/combo-strategy.txt`.
- If no strategy is saved yet, it uses `general.bat`.

Run `pick-strategy.cmd` as Administrator to test all `general*.bat`
ALT/FAKE variants and save the best one for `zapret-all.cmd`.

Close the window or press `Q` to stop zapret. The launcher and
`utils/combo-watchdog.ps1` close `winws.exe` and `TgWsProxy_windows.exe` when
zapret exits or the window is closed.

On the first run `zapret-all.cmd` downloads `TgWsProxy_windows.exe` from
Flowseal releases into the ignored runtime folder `tg-ws-proxy/`.

Telegram Desktop setup:

- Use the TG WS Proxy tray menu item `Open in Telegram`, or copy the proxy link.
- Manual settings: MTProto, server `127.0.0.1`, port `1443`.
- The secret is generated by TG WS Proxy and shown in its first-run window,
  tray link, and logs.

Extra modes:

- `zapret-all.cmd download-telegram` downloads TG WS Proxy without starting it.
- `zapret-all.cmd telegram-only` starts only TG WS Proxy.
- `zapret-all.cmd stop` stops zapret and TG WS Proxy.

SoundCloud is included through `soundcloud.com` and `sndcdn.com` in
`lists/list-general.txt`.
